Introduçāo ao Git e GitHub

Github é uma plataforma colaborativa desenvolvedor online (rede social para os desenvolvedores) que serve como espaço de armazenamento on-line onde os engenheiros compartilham seus projetos

Git é um software que roda no centro do GitHub é um software de controle de versão, o que significa que podemos fazer mudanças em um projeto sem substituir qualquer parte desse projeto. É uma maneira de interagindo com os arquivos em sua máquina local e ser capaz de atualizar os arquivos armazenados no Github. Estaremos usando Git e Github em todos os nossos meetups,você estará recebendo e enviando todos os desafios de código através GitHub, por isso é fundamental criar a sua conta no GitHub e aprender alguns passos básicos, segue abaixo algumas instruções e links para estudo.

  • Tutorial Comandos Básico GitHub
  • Codecademy - Learn Git

    Curso de apenas 2 horas, vai ensinar tudo que você precisa saber para começar a usar o Git imediatamente. Certifique-se de praticar em sua própria máquina (fazer alguns diretórios apenas para praticar) para garantir que você está confortável com o uso do Git.

  • The Git Game

    Seleciona o nível do game e aprenda Git de uma forma divertida! :)

Formato Meetup Online

  1. Introdução de conceitos/ slides ( 30 minutos)
  2. Atividade prática - Desafio do código: (1 hora) podendo dividir em pares ( pair programing).
  3. No final de cada meetup, solicitar para os participantes publicarem o seu exercício no repositório do WWcode.
  4. Todo o material do grupo de estudo estará disponível na página do GitHub.
  5. Os encontros serāo semanais, 1 vez por semana (quarta-feira - 19:30 às 21hs) caso ocorra alteraçāo no calendário, iremos informar com antecedência através de e-mail e mídias sociais.
  6. Junte-se a nós na plataforma Meetup.com e fique por dentro de todos os nosso eventos.

Meetup Presencial

  • Fechamos uma parceria com a nós coworking (marco zero) que está apoiando com o local para a realizaçāo dos meetups presenciais, os encontros serāo realizados uma vez por mês (quinta ou sexta) no horário: 18:30 - 21hs.
  • Este evento é um complemento dos meetups onlines, a ideia é desenvolver pequenos projetos(formulário,sites e apps)colocar a māo na massa! e claro, nos conhecermos pessoalmente,fazer novas amizades e network!
  • Um dos nosso objetivos é desenvolver projetos para organizações sem fins lucrativos paralelo ao grupo de estudos, acreditamos que é uma ótima oportunidade para elaborar um portfólio e adquirir experiência em projetos reais.

    Wome Who Code

    Women Who Code é uma organizaçāo global sem fins lucrativos que inspira mulheres na carreira de tecnologia, com um conjunto de programas para os engenheiros para ajudar a construir as carreiras que queremos. Trabalhamos para apoiar esta geração tornar-se líderes e modelos na indústria de tecnologia,com sede em San Francisco-CA, presente em 20 países, 60 cidades e 80k membros.
